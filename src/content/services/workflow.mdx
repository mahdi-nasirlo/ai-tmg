# ماژول ورک‌فلو

این ماژول به شما امکان می‌دهد گردش کارهای سازمانی را با گراف‌های بصری طراحی، اجرا و مانیتور کنید.

## ویژگی‌ها

- طراحی گردش کار با Drag & Drop
- تعریف شرایط و شاخه‌های تصمیم‌گیری
- ثبت تاریخچه اجرای فرآیندها
- ارتباط مستقیم با فرم‌ساز و سیستم اعلان

## مثال

```js
const workflow = createWorkflow({
  name: "فرآیند تأیید فاکتور",
  steps: ["بررسی اولیه", "تأیید مالی", "ارسال نهایی"],
});
```
